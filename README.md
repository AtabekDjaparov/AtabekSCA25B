Обратный порядок слов в предложении

Простая консольная программа на Python, которая принимает от пользователя предложение и выводит его слова в обратном порядке


Особенности

- Обрабатывает лишние пробелы
- Предусмотрена защита от ввода пустых строк
- Удобный выход по команде `выход`


Как запустить

1. Убедитесь, что у вас установлен Python 3
2. Запустите файл `main.py` через терминал (Win + R, cmd):
   python atabek.py


В случае если терминал или редактор не находит ваш файл:

1. Так же откройте терминал
2. Напишите python (можно и py или же python3) и оставьте после него пробел
3. Найдите в проводнике скачанный файл atabek.py и мышкой перетащите его в терминал
4. Нажмите Enter
